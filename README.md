# IoT 기반 지반 및 시설물 계측 통합 모니터링 플랫폼 (DainMonitoring)

이 프로젝트는 국가 중요 문화재(경주 첨성대, 경복궁 근정전, 공주 마곡사 등) 및 주요 도심 건설 현장(서초동, 마곡지구 등)의 지반 붕괴, 균열, 변위, 기울기 등을 실시간으로 수집하고 모니터링하여 재난 위험을 예방하는 **Spring Boot 기반 IoT 통합 계측 관제 솔루션**입니다.

분산된 현장 장비(LTE 로거 및 계측 PC)로부터 전달되는 원천 데이터 수집 파이프라인을 구축하고, 멀티 테넌시 배포 아키텍처 개편 및 수치 연산 정합성 보장을 통해 시스템의 가용성과 안정성을 획기적으로 개선했습니다.

---

## 🛠️ 기술 스택
* **Framework**: Spring Boot (Java 17)
* **Build Tool**: Maven (pom.xml)
* **Database**: MariaDB, MyBatis (Mapper)
* **Infrastructure**: Nginx, Apache Tomcat, Linux (CentOS)
* **APIs**: Kakao Maps API (위치 매핑), SMS 발송 API (긴급 알림)

## 🏗️ 핵심 시스템 구조 및 주요 성과

### 1) Nginx 리버스 프록시 기반 단일 배포 아키텍처 (멀티 테넌시)
* **Problem**: 기존 시스템은 새로운 계측 현장(현장 코드별)이 추가될 때마다 독립된 WAS 인스턴스를 하나씩 별도 구축하고 WAR 파일로 개별 배포하여, 현장 수가 증가할수록 배포 오버헤드와 서버 리소스 낭비가 매우 극심했습니다.
* **Solution**: 단일 JAR 패키징 배포 구조로 통합하고, **Nginx 리버스 프록시(Reverse Proxy)**를 라우터로 활용하여 단일 WAS 서버 내에서 멀티 테넌시(Multi-Tenancy) 형태로 동적 라우팅되도록 개선했습니다.
* **Outcome**: 신규 현장 개설 시 배포 포인트를 하나로 단일화하여 **유지보수 효율을 200% 향상**시켰으며, 서버 운영 비용을 획기적으로 절감했습니다.

### 2) Decimal 데이터 표준화를 통한 연산 오차 0% 달성
* **Problem**: 지반/구조물의 미세한 균열 변위(단위: mm)를 표현할 때, 기존 비정규화된 문자열(String) 타입 적재나 float/double 타입의 수치형을 활용하여 다차원 누적 연산 시 **부동 소수점 오차**가 누적되어 정확한 위험 임계치 판정에 오차가 발생했습니다.
* **Solution**: 데이터 수집부부터 데이터베이스 DDL 타입까지 **Decimal 데이터 타입 표준화**를 선언하고 백엔드 연산 시 `BigDecimal` 클래스를 엄격하게 활용하여 고정소수점 정밀 연산을 수행하도록 리팩토링했습니다.
* **Outcome**: 미세 변위 연산 오차 0%를 달성하여 시스템 수치 데이터 정합성 100%를 보장하고 오경보율을 제거했습니다.

### 3) Kakao Maps API 연동 실시간 변위 시각화
* **Problem**: 단순 텍스트 표 형태의 관제 화면은 재난 발생 시 정확한 위험 발생 지점을 시각적으로 직관적으로 인지하기 어려웠습니다.
* **Solution**: **Kakao Maps API**를 연동하여 지도 상에 각 계측 센서의 실제 위경도 좌표를 실시간 동기화하고 매핑했습니다. 
* **Outcome**: 센서가 보내는 변위 데이터를 4단계(보통, 주의, 경고, 위험)의 위험 임계치별 색상과 애니메이션 효과로 매핑하여 관제사의 긴급 재난 상황 시각적 인지 속도를 대폭 가속했습니다.

### 4) 비동기 긴급 장애 경보 알림 시스템 (SMS API 연동)
* **Problem**: 붕괴 위험 감지 시 관리자에게 즉각적으로 경보를 발송해야 했으나, 동기식 호출 방식을 사용할 경우 통신사 SMS API 지연으로 인해 수집 스레드가 멈추거나 다음 계측 수신이 밀리는 현상이 발생했습니다.
* **Solution**: 스프링 비동기 실행 스레드 풀을 구성하여 임계치 초과 데이터 감지 시 **SMS API 발송 로직을 비동기(Asynchronous)로 완벽히 분리** 처리했습니다.
* **Outcome**: 계측 데이터 적재 가용성을 보장하는 동시에, 1초 이내에 안전 관리 실무자 전원에게 실시간 긴급 경보 문자가 도달하는 신속한 장애 복구 프로세스를 수립했습니다.

* **지자체 건설 및 마곡 현장** (`seochodong`, `ddm`, `jungdong` 등): 토류판 기울기 및 지반 침하 실시간 추적
